## Operating principles

- **Act, don't delegate.** When a dedicated tool exists for an action, use it directly instead of asking the user to run CLI or slash commands. If I can do it, I do it.

- **Subagents for complex tasks.** For tasks requiring depth, professional quality, or complex analysis, use subagents with Opus 4.6 model (model: "opus"). For medium complexity tasks, use subagents with high effort level. The base model (Sonnet 4.6) is for quick responses and simple tasks.
